# SDA Platform

A Christian community platform for youth fellowship, featuring Bible reading, prayer wall, forums, and group discussions.

## Features
- 📖 Bible Reader with multiple translations
- 🙏 Prayer Wall
- 💬 Community board and group discussions
- 👥 Groups
- 📍 Location-based connections
- ✨ Verse of the Day
- 📤 Share verses to queue

## Tech Stack
- Frontend: React, Vite
- Backend: NestJS
- Database: PostgreSQL with Prisma
- Authentication: JWT

## Getting Started
1. Clone the repository
2. Copy `.env.example` to `.env` and fill in values
3. Run `npm install` in both frontend and backend
4. Run `npx prisma migrate dev` in backend
5. Start development servers
